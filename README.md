# DTwinBackend
硕士毕业论文系统后端代码

# 主体功能

1. 连接装配实验平台，通过UDP监听端口数据
2. 定义各设备的xml信息模型
3. 根据各设备信息模型解析xml，存入redis中与mysql中
4. 根据信息模型，定义接口，从redis中提供前台数字孪生模型相应实时运动数据
5. 定义后台mysql接口，对装配过程数据进行管理
